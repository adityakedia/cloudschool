Important Steps for Updating the Latest Version

1. Switch Environment to Local
Before uploading a new addon, ensure that you change your environment from Production to Local. This step is crucial to prevent any potential issues with the live system. For detailed instructions, refer to the https://help.techvill.net/switching-from-production-to-local-environment/.

2. Verify Upgrader Version
Make sure you have the latest upgrader version (V1.3.0 or above) installed before proceeding with the update process. The latest version can be found https://techvill.net/artifism/plugins?product=artifism.

Proceed with the Latest Update
After completing the above steps, you can proceed with applying the latest update to your add-on.
